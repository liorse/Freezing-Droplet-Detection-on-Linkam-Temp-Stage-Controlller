How to install WISDOM software
installation files located here:
C:\Users\naamarei\Documents\WISDOM_Installation_files

1. Install Labview 2015 SP1 - 2015sp1LV-WinEng.zip
2. Install device drivers - keep default checked - Aug_2015_Device_Drivers.zip
3. Install Vision software - VISION_2015SP1.zip
4. Install multicore analysis ni - 
https://www.ni.com/en-il/support/downloads/tools-network/download.labview-multicore-analysis-and-sparse-matrix-toolkit.html#374384
5. Install openg and mgi packages from vipm.
openg version should be 4.0.x
